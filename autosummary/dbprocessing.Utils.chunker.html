dbprocessing.Utils.chunker¶

dbprocessing.Utils.chunker(seq, size)[source]¶

Return a long iterable in a tuple of shorter lists. Taken from http://stackoverflow.com/questions/434287/what-is-the-most-pythonic-way-to-iterate-over-a-list-in-chunks

Parameters:
  • seq (iterable) – Iterable to split up
  • size (int) – Size of each split in the output, last one has the remaining elements of seq
Returns:

A tuple of lists of the iterable seq split into len(seq)/size segments
Return type:

tuple
