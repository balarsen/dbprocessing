dbprocessing.DBfile.DBfile¶

class dbprocessing.DBfile.DBfile(diskfile, dbu, makeDiskFile=False)[source]¶

DBfile class is an extension of Diskfile that takes a physical file on disk and maps it to the database file entry, this is not mapped to te file table in the DB but instead a bridge between the two.
